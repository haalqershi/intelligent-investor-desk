# Intelligent Investor Desk

A full-stack, event-driven wealth management platform built with microservices architecture.

## Architecture
- **Backend**: Java 17, Spring Boot 3.2, Spring Cloud
- **Frontend**: Angular 18
- **Data**: PostgreSQL, Redis
- **Messaging**: RabbitMQ
- **Monitoring**: Prometheus, Grafana, Jaeger

## Getting Started
See [docs/SETUP.md](docs/SETUP.md) for detailed setup instructions.

## Quick Start
```bash
cd infrastructure\docker
docker-compose up -d
```
